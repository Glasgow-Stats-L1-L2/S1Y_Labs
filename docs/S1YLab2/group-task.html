3.4 Q4

ggplot(cdc,aes(x=gender,y=wdiff))+ geom_boxplot()

It appears on average there is a slightly greater difference in current and desired weight for women than for men, though this is difficult to see from the box plot alone. Numerical summaries could be used to clarify this like so:

cdc.men<-cdc%>% 
  filter(gender=="m") 

cdc.women<-cdc%>% 
  filter(gender=="f") 

summary(cdc.men$wdiff)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
## -300.00  -20.00   -5.00  -10.71    0.00  500.00
summary(cdc.women$wdiff)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
## -300.00  -27.00  -10.00  -18.15    0.00   83.00

However to check rigorously if this result is significant, we would need to conduct a hypothesis test, which we will cover later in the course.
